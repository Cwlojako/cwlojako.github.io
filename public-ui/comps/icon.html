# Icon 图标

提供了一套常用的图标集合。

# 使用方法

直接通过设置类名为 cw-icon-iconName 来使用即可。例如：

搜索
<i class="cw-icon-service"></i>
<i class="cw-icon-help"></i>
<i class="cw-icon-bell"></i>

<cw-icon name="service"></cw-icon>
<cw-icon name="help"></cw-icon>
<cw-icon name="bell"></cw-icon>
<cw-button type="primary" icon="cw-icon-search">搜索</cw-button>

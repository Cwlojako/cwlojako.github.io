    # 禁用状态

    按钮不可用状态。

    默认按钮 主要按钮 成功按钮 信息按钮 警告按钮 危险按钮

    你可以使用disabled属性来定义按钮是否可用，它接受一个Boolean值。

    <cw-button disabled>默认按钮</cw-button>
<cw-button type="primary" disabled>主要按钮</cw-button>
<cw-button type="success" disabled>成功按钮</cw-button>

    # 图标按钮

    带图标的按钮可增强辨识度（有文字）或节省空间（无文字）。

    搜索 上传

    设置icon属性即可，icon 的列表可以参考 Quickwork 的 icon 组件，也可以设置在文字右边的 icon ，只要使用i标签即可，可以使用自定义图标。

    <cw-button type="primary" icon="cw-icon-yiwen"></cw-button>
<cw-button type="primary" icon="cw-icon-shu"></cw-button>
<cw-button type="primary" icon="cw-icon-zan"></cw-button>
<cw-button type="primary" icon="cw-icon-chaping">搜索</cw-button>
<cw-button type="primary"
  >上传<i class="cw-icon-tupian cw-icon--right"></i
></cw-button>

    # 长按钮

    设置属性 block 可将按钮的宽度设置为 100%，常用于弹窗内操作按钮。除此之外，使用者也可以直接通过给组件添加 style 来设置更加细节的样式，比如定宽。

    Primary Success Info Warning Error
    <cw-row class="demo-button-block">
  <cw-button type="primary" block>Primary</cw-button>
  <cw-button type="success" block plain>Success</cw-button>
  <cw-button type="info" block>Info</cw-button>
  <cw-button type="warning" block plain dashed>Warning</cw-button>
  <cw-button type="error" block>Error</cw-button>
</cw-row>

    # 加载中

    点击按钮后进行数据加载操作，在按钮上显示加载状态。

    加载中

    要设置为 loading 状态，只要设置loading属性为true即可。

    <cw-button type="primary" :loading="true">加载中</cw-button>
    Expand Copy Copy

    # 不同尺寸

    Button 组件除了默认尺寸外，还可以根据场景选择 medium 和 small 尺寸。

    默认按钮 中型按钮 小型按钮

    通过设置size属性来配置它们。

    <cw-button type="primary">默认按钮</cw-button>
<cw-button type="primary" size="medium">中型按钮</cw-button>
<cw-button type="primary" size="small">小型按钮</cw-button>
    Expand Copy Copy

    # Attributes

    参数 说明 类型 可选值 默认值
    size 尺寸 String medium / small —
    type 类型 String primary / info / error / success —
    plain 是否朴素按钮 Boolean — false
    dashed 是否虚线按钮 Boolean — false
    text 是否文字按钮 Boolean — false
    round 是否圆角按钮 Boolean — false
    circle 是否圆形按钮 Boolean — false
    loading 是否加载中状态 Boolean — false
    loading-size 加载指示器大小 Number — 14
    loading-color 加载指示器颜色 String — #FFFFFF
    disabled 是否禁用状态 Boolean — false
    block 是否长按钮 Boolean — false
    icon 图标类名 String — —
    native-type 原生 type 属性 String button / submit / reset button

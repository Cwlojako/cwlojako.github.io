# Transition 动画

cwPublicUi 内置了很多种过渡动画，其内部组件的动效也都是基于这些动画，因此，我们特地封装了 Transition 组件，让你也可以使用。在使用之前请仔细阅读 transition 组件文档

# 淡入淡出

Click Me
.cw-fade-in-linear
Click Me
.cw-fade-in

提供 cw-fade-in-linear 和 cw-fade-in 两种效果

<template>
  <cw-button @click="show1 = !show1">Click Me</cw-button>
  <div class="container">
    <cw-transition name="cw-fade-in-linear">
      <div v-show="show1" class="transition-item">.cw-fade-in-linear</div>
    </cw-transition>
  </div>
  <cw-button @click="show2 = !show2">Click Me</cw-button>
  <div class="container">
    <cw-transition name="cw-fade-in">
      <div v-show="show2" class="transition-item">.cw-fade-in</div>
    </cw-transition>
  </div>
</template>

<script>
  export default {
    data() {
      return {
        show1: true,
        show2: true,
      };
    },
  };
</script>

# 展开折叠

使用 cw-collapse-transition 组件实现折叠展开效果

Click Me
cw-collapse-transition
cw-collapse-transition
<template>
  <cw-button @click="show = !show">Click Me</cw-button>
  <div class="container" style="height: 200px;">
    <cw-collapse-transition>
      <div v-show="show">
        <div class="transition-item">cw-collapse-transition</div>
        <div class="transition-item">cw-collapse-transition</div>
      </div>
    </cw-collapse-transition>
  </div>
</template>

<script>
  export default {
    data() {
      return {
        show: true,
      };
    },
  };
</script>

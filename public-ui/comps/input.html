    # Input 输入框

    通过鼠标或键盘输入字符

    # 基础用法

    <template>
  <cw-input v-model="input" placeholder="请输入内容" :width="280"></cw-input>
</template>

<script>
  export default {
    data() {
      return {
        input: ""
      }
    }
  }
</script>

    # 可自适应文本高度的文本域

    通过设置 autosize 属性可以使得文本域的高度能够根据文本内容自动进行调整，并且 autosize 还可以设定为一个对象，指定最小行数和最大行数。

    <template>
  <cw-input
    type="textarea"
    autosize
    placeholder="请输入内容"
    v-model="textarea1"
    :width="400"
  ></cw-input>
  <div style="margin: 20px 0;"></div>
  <cw-input
    type="textarea"
    :autosize="{ minRows: 3, maxRows: 5 }"
    placeholder="请输入内容"
    v-model="textarea2"
    :width="400"
  ></cw-input>
</template>

<script>
  export default {
    data() {
      return {
        textarea1: "",
        textarea2: ""
      }
    }
  }
</script>

    # 输入长度限制

    maxlength 和 minlength 是原生属性，用来限制输入框的字符长度，其中字符长度是用 Javascript 的字符串长度统计的。对于类型为 text 或 textarea 的输入框，在使用 maxlength属性限制最大输入长度的同时，可通过设置 showMaxLength 属性来展示字数统计。

    <cw-input
    type="text"
    placeholder="请输入内容"
    v-model="text"
    maxlength="10"
    show-max-length
    :width="280"
></cw-input>
<div style="margin: 20px 0;"></div>
<cw-input
    type="textarea"
    placeholder="请输入内容"
    v-model="textarea"
    maxlength="30"
    show-max-length
    :width="400"
></cw-input>

<script>
  export default {
    data() {
      return {
        text: "",
        textarea: ""
      }
    }
  }
</script>

    # Attributes

    参数 说明 类型 可选值 默认值
    type 类型 String text，textarea，password —
    value / v-model 绑定值 String / Number — —
    width 输入区域宽度 String / Number — 100%
    maxlength 原生属性，最大输入长度 Number — —
    minlength 原生属性，最小输入长度 Number — —
    show-max-length 是否显示输入字数统计，只在 type=text 或 type=textarea 时有效 Boolean — false
    placeholder 输入框占位文本 String — —
    clearable 是否可清空 Boolean — false
    show-password 是否显示切换密码图标 Boolean — false
    disabled 禁用 Boolean — false
    size 输入框尺寸，只在 type!=textarea 时有效 String small —
    prefix-icon 输入框头部图标 String — —
    suffix-icon 输入框尾部图标 String — —
    rows 输入框行数，只对 type=textarea 有效 Number — 2
    autosize 自适应内容高度，只对 type=textarea 有效，可传入对象，如，{ minRows: 2, maxRows: 6 } Boolean / Object — false
    autocomplete 原生属性，自动补全 String on / off off
    name 原生属性 String — —
    readonly 原生属性，是否只读 Boolean — false

    # Slots

    name 说明
    prefix 输入框头部内容
    suffix 输入框尾部内容
    prepend 输入框前置内容
    append 输入框后置内容

    # Events

    事件名称 说明 回调参数
    blur 在 Input 失去焦点时触发 (event: Event)
    focus 在 Input 获得焦点时触发 (event: Event)
    change 在 Input 值改变时触发（与原生统一，失焦时触发） (value: String | Number, event: Event)
    input 在 Input 值改变时触发 (value: String | Number, event: Event)
    clear 在点击由 clearable 属性生成的清空按钮时触发 —

    # Methods

    方法名 说明 参数
    focus 使 input 获取焦点 —
    blur 使 input 失去焦点 —
    select 选中 input 中的文字 —
